## Теоретическая база

### Медианный фильтр
Медианный фильтр — нелинейный фильтр для подавления шумов на изображениях. В отличие от линейных фильтров (гауссов, усредняющий), он сохраняет резкие границы объектов.

**Принцип работы:**
- Для каждого пикселя рассматривается окрестность размером k×k
- Вычисляется медиана значений пикселей в этой окрестности
- Исходное значение пикселя заменяется на медиану

**Преимущества:**
- Эффективно удаляет импульсный шум (шум "соль-перец")
- Сохраняет четкие границы объектов
- Устойчив к выбросам

### Алгоритмы
Реализованы две версии медианного фильтра:

1. **Библиотечная реализация** - `cv2.medianBlur()`
2. **Нативная реализация**

### Архитектура
- **median_filter_native()** - нативная реализация с обработкой границ через отражение
- **process()** - функция сравнения производительности
- **visualise()** - функция визуализации результатов

### Особенности реализации
- Обработка цветных изображений (3 канала)
- Корректная обработка границ через padding с отражением
- Поддержка различных размеров ядра (5×5, 9×9, 13×13, 17×17)
